static关键字

发表于 2018-04-01 | 分类于 java ， java面向对象

static可以声明属性和方法

1.static定义属性：用static声明的变量，直接用类名调用（类名.变量）。

2.static定义方法: 直接用类.方法()调用。

总结：

①：static方法只能调用static属性和方法;

②：非static方法不受任何限制；

改进———————->

所有的非static方法必须由对象调用，因此static方法如果要想使用非static操作，必须产生对象才能调用
