javascript正则表达式

发表于 2018-01-30 | 分类于 JavaScript ， 正则表达式

1 .修饰符
g：global全文搜索，不添加g，搜索到第一个匹配停止。
i：ignore case忽略大小写，默认大小写敏感。
He is a boy. Is he? .replace(/\bis\b/g,'0')
He 0 is a boy. Is he?

He is a boy. Is he? .replace(/\bis\b/gi,'0')
He 0 is a boy. 0 he?
2 .字符类
表达式[abc]把字符a或b或c归为一类。
'a1b2c3d4'.replace(/[abc]/g,'X')—->X1X2X3d4
表达式[^abc]表示不是字符a或b或c的内容。
'a1b2c3d4'.replace(/[^abc]/g,'X')—->aXbXcXXX
3 .范围类
[a-z]来连接两个字符表示从a到z的任意字符。这是闭区间，包含a到z本身。
'a1b2c3x4d9'.replace(/[a-z]/g,'Q')—>'Q1Q2Q3Q4Q9'
[a-zA-Z]
'a1b2c3x4d9CDERFV'.replace(/[a-zA-Z]/g,'Q')—>'Q1Q2Q3Q4Q9QQQQQQ'
[0-9]
'2015-09-06'.replace(/[0-9]/g,'A')—>"AAAA-AA-AA"(带横线)
'2015-09-06'.replace(/[0-9-]/g,'A')—>"AAAAAAAAAA"（不带横线）
4 .预定义类

匹配一个ab+数字+任意字符的字符串—->ab\d.
5 .边界

'This is a boy'.replace(/is/g,'0')—>"Th0 0 a boy"（不带单词边界\b的）
'This is a boy'.replace(/\bis\b/g,'0')—>"This 0 a boy"（带单词边界\b的）
'This is a boy'.replace(/\Bis\b/g,'0')—>"Th0 is a boy"（观察前面无边界后面有边界）

'@123@abc@'.replace(/@./g,'Q')—>"Q23Qbc@"（替换@和任意字符）
'@123@abc@'.replace(/^@./g,'Q')—>"Q23@abc@"（替换以@和任意字符开头的）
'@123@abc@'.replace(/.@/g,'Q')—>"@12QabQ"（替换任意字符和@的）
'@123@abc@'.replace(/.@$/g,'Q')—>"@123@abQ"（替换任意字符和@结束的）
6 .量词

7 .分组
使用（）可以达到分组的效果。
'a1b2c3d4'.replace(/[a-z]\d{3}/g,'X')—>"a1b2c3d4"（不带分组的）
'a1b2c3d4'.replace(/([a-z]\d){3}/g,'X')—>"Xd4"（带分组的）
8.反向引用
'2015-02-05'.replace(/\d{4}-\d{2}-\d{2}/g,'$1')—>"$1"
'2015-02-05'.replace(/(\d{4})-(\d{2})-(\d{2})/g,'$1')—>"2015"
'2015-02-05'.replace(/(\d{4})-(\d{2})-(\d{2})/g,'$2')—>"02"
'2015-02-05'.replace(/(\d{4})-(\d{2})-(\d{2})/g,'$3')—>"05"
'2015-02-05'.replace(/(\d{4})-(\d{2})-(\d{2})/g,'$2/$3/$1')—>"02/05/2015"（将年-月-日转换为月/日/年）
9 .test方法：用于测试字符串参数中是否存在匹配正则表达式模式的字符串。
